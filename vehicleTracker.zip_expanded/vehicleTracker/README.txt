
###Database:
	Find the database schema inside "dbSchema" folder
	PATH: vehicleTracker.zip_expanded\vehicleTracker\dbSchema\vehicletracker.sql

###Technology used:
	Eclipse  Java IDE
  
   Dependencies:
	SpringMVC
	Spring Data JPA
	ThymeLeaf
	Hibernate
	MySQL (Please note: MySql Server: 8.0)


###### HOW TO RUN ######

Linux:
`./mvnw spring-boot:run` and go to a browser at [http://localhost:8080](http://localhost:8080).

Windows:
`./mvnw.cmd spring-boot:run` and go to a browser at [http://localhost:8080](http://localhost:8080).


OR

 Open from your IDE as a Maven project. 
 Then "Run as Java Application"  
	- VehicleTrackerApplication.java
